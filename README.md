# Telco Customer Churn Prediction

## Dataset Source
IBM Telco Customer Churn Dataset from Kaggle:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

Download the dataset and place this CSV file in the same folder as the notebook:

`WA_Fn-UseC_-Telco-Customer-Churn.csv`

## Project Overview
This project predicts whether a telco customer is likely to churn using machine learning.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Models Used
- Logistic Regression
- Random Forest Classifier

## Resume Description
Developed a customer churn prediction model using the IBM Telco Customer Churn dataset. Performed data cleaning, exploratory data analysis, categorical encoding, model training, and evaluation using Logistic Regression and Random Forest.
